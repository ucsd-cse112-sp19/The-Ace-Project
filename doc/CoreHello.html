Class: CoreHello

CoreHello()

A custom element that renders Hello World followed by whatever children it is given.
It can also accept the attributes lang to change the Hello World language and rainbow for extra swag

Constructor

new CoreHello()

Properties:
Name Type Attributes Default Description
lang string <optional>
 "en" Language of Hello World - Accepts "en" | "jp" | "pt"
rainbow boolean <optional>
 false If present, animates the text with a rainbow effect

Example
<core-hello rainbow lang="pt"> Joseph </core-hello>
